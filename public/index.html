Image Search Abstraction Layer

This project is from FreeCodeCamp's Back End Certification curriculum. It takes a valid url and returns a JSON object with the original url and a hex string that represents a shortened version of the url. When you pass a valid hex (one that already exists in the database), the app will redirect you to the associated original url.

Example Usage

https://SOMETHING.herokuapp.com/google.com

Example Output
